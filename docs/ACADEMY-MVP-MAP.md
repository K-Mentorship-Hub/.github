# Academy & MVP Map

## Why This Page Exists

This page groups the hub into practical directions that people can immediately understand.

The point is not to make everyone fit into one label forever.

The point is to make it easy to say:

- where am I starting?
- what kind of path fits me?
- what kind of MVP direction could grow from this?

---

## The Main Hub Logic

Every direction can show up in:

- `Academy` — learn the fundamentals and get practical guidance
- `MVP Lab` — turn the direction into an idea, prototype, or validation path
- `Networking` — meet people in the same or adjacent lane

So instead of treating `Academy`, `MVP`, and `Networking` as separate worlds, the hub uses one direction map across all three.

## Best Practical Use

If you are joining the hub:

1. choose your `primary sphere`
2. choose your `direction`
3. choose your `main goal`
   - `Academy`
   - `MVP Lab`
   - `Networking`
4. use the hybrid label only if it really helps

That is enough to start.
