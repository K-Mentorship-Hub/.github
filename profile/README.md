# K Mentorship Hub

**Continuous Learning, Continuous Integration.**

`K Mentorship Hub` is the practical part of the wider `SET` ecosystem by Oksana Kolisnyk.

It helps people:

- learn through guided paths
- build practical MVPs and public cases
- connect across science, entrepreneurship, and technology
- move from confusion to a clear next step

---

## The `SET` Logic

- `S = Science`
- `E = Entrepreneurship`
- `T = Technology`

These are not three isolated worlds.
They are a navigation system.

The hub supports:

- `S`
- `E`
- `T`
- `S+T`
- `S+E`
- `E+T`
- `S+E+T`

- `SPHERE-II-ENTREPRENEURSHIP` -> ventures, market logic, founder systems, validation, and public cases
- `SPHERE-III-TECHNOLOGY` -> tools, code, AI, dashboards, infrastructure, and reusable systems

## Practical Layers

`K Mentorship Hub` works best as a layered system:

- GitHub -> public docs, repositories, contribution surface
- Discord -> live interaction and community routing
- Notion -> guidebook, intake, and operating layer
- Hugging Face -> demos and runtime interfaces when needed

---

## How To Read Your Direction

Use this shortcut:

- choose `S` if your main question is about understanding, measuring, testing, or investigating
- choose `E` if your main question is about building, validating, positioning, growing, or organizing
- choose `T` if your main question is about code, tools, systems, infrastructure, or implementation

If your work spans more than one sphere, that is normal.
